## Installation

1. Just clone this repo:

```bash
git clone git@github.com:mittelgrau/mittelgrau-kirby kirby-starter
```

2. Go into your new project directory:

```bash
cd kirby-starter
```

3. Install kirby and all vendors by:

```bash
composer install
```

4. Install all required NPM packages by:

```bash
pnpm install
```
or
```bash
npm install
```
